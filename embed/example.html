CodexPet 嵌入 SDK

把 embed/codex-pet.js 引入任意网页，配置宠物目录 URL 或雪碧图 URL 即可显示。 本页需通过本地服务器打开： python -m http.server 8765 → http://localhost:8765/embed/example.html

方式 A · data 属性自动挂载（最省事）

<!-- 页面任意位置放一个容器 -->
<div
  data-codex-pet
  data-pet-json="../firefly/pet.json"
  data-spritesheet="../firefly/spritesheet.webp"
  data-state="idle"
  data-scale="1"
  data-speed="100"
  data-position="bottom-right"
  data-mode="fixed"
  data-draggable="true"
></div>

<script src="./codex-pet.js"></script>

下方右下角浮层就是方式 A 挂载的「流萤」（fixed 全页浮层）。

方式 B · JS API 手动挂载（可动态控制）

const pet = CodexPet.mount('#box', {
  // 推荐：两个路径分开配置
  petJson: '../ganyu-pet-v2/pet.json',
  spritesheet: '../ganyu-pet-v2/spritesheet.webp',
  // 也支持远程完整 URL：
  // petJson: 'https://cdn.example.com/pet.json',
  // spritesheet: 'https://cdn.example.com/sheet.webp',
  mode: 'absolute',          // 限制在 #box 容器内
  position: 'center',
  scale: 1,
  speed: 100,
  draggable: true,
  onReady(inst) { console.log(inst.listStates()); },
});

pet.setState('waving');
pet.setScale(1.2);
pet.load({
  petJson: '../firefly/pet.json',
  spritesheet: '../firefly/spritesheet.webp',
});

方式 C · 只给图片 URL（无 pet.json）

CodexPet.mount(document.body, {
  petJson: 'https://your.cdn.com/pets/firefly/pet.json',
  spritesheet: 'https://your.cdn.com/pets/firefly/spritesheet.webp',
  // petJson 可不传，此时使用内置 8×9 默认动画表
  mode: 'fixed',
  position: 'bottom-left',
  scale: 0.9,
});

跨域图片必须在 CDN 配置 Access-Control-Allow-Origin， 否则 canvas 无法裁切绘制。

配置项一览

petJson       pet.json 的 URL/路径，或直接传入对象   ← 路径 1
spritesheet   spritesheet.webp 的 URL/路径           ← 路径 2
src           （可选兼容）宠物目录，自动拼上面两个文件
state         初始动画，默认 idle
scale         显示倍率，默认 1
speed         毫秒/帧，默认 100
position      bottom-right | bottom-left | top-right | top-left | center | {x,y}
mode          fixed（浮在整页）| absolute（容器内）| inline（文档流）
draggable     是否可拖，默认 true
clickCycle    单击切换动画，默认 true
dragAnim      拖动时左右跑，默认 true
releaseState  松手后回到的状态，默认 idle；设 null 保持
zIndex        层级，默认 2147483000
margin        角落边距，默认 16
visible       是否显示
paused        是否暂停
flip          水平镜像
onReady / onError / onStateChange  回调

在你自己的任务页里最小接入

<!-- 任务系统 HTML -->
<div id="task-pet"></div>
<script src="/static/codex-pet.js"></script>
<script>
  // 配置来自后台 / 表单均可
  window.taskPet = CodexPet.mount('#task-pet', {
    petJson: '/pets/firefly/pet.json',
    spritesheet: '/pets/firefly/spritesheet.webp',
    mode: 'fixed',
    position: 'bottom-right',
    scale: 1,
    state: 'idle',
  });

  // 任务状态联动示例
  function onTaskStart()  { taskPet.setState('running'); }
  function onTaskWait()   { taskPet.setState('waiting'); }
  function onTaskFail()   { taskPet.setState('failed'); }
  function onTaskDone()   { taskPet.setState('review'); }
</script>
